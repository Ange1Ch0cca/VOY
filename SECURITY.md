## Reporting a Vulnerability

If you discover a security vulnerability affecting the **VOY** platform or any component of this repository, please report it privately.

Do **not** disclose security vulnerabilities through GitHub Issues, Pull Requests, or any public communication channel.

The **VOY** platform is developed and maintained by **UNIT**. All security reports should be submitted directly to the development team.

**Contact:** +51906829934

**Email:** security@unit.pe

All reported vulnerabilities will be reviewed, validated, and addressed as soon as possible.

We appreciate your responsible disclosure and your contribution to keeping the platform secure.
